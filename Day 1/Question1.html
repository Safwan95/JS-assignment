Console

Console is mainly used for debugging and is accessed from console tab in the browser. In chrome press F12 to access the console. Types of consoles are:
-> console.log()
-> console.warn()
-> console.error()
-> console.assert()
-> console.clear()
-> console.count()
-> console.debug()
-> console.dir()
-> console.group()
-> console.groupEnd()
-> console.info()
-> console.table()
-> console.time()
-> console.timeEnd()
-> console.timeLog()

console.log

It is used generally for display the output. The value to be displayed is to be written in the bracket.

console.warn

It is used to output warning messages. It will be highlighted in red colour in the background.

console.error

It is used to output error messages. It will be highlighted in yellow colour in the background.

console.assert

It logs an error message to the console if a given argument is false. If the argument is true then nothing happens.

console.clear

It is used to clear the console. Just clear can be also used instead.

console.count

It logs the number of times this particular is called.

console.debug

It used to output the message debug to the debug log level.

console.dir

It is used to see all the properties of a specified JS object. By using this we can easiy access all the properties of the object.

console.group

It indicates the start of a message group. After using this, things written in this will be considered in the group.

console.groupEnd

It is used to the group created by using console.group.

console.info

It is used to output informational messages to the web consoles.

console.table

It is used to display the data in tabular form or it is used create a table for the data.

console.time

It is used to start a timer in console in the browser.

console.timeEnd

It is used to end the timer started using console.time .

console.timeLog

It is used to log the value of the specified timer in to the console.
