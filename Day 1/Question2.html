Var

It is used to create or declare a variable. The syntax is var "variable name".It can be redeclared and updated. It is funtion scoped , that is they are available only in the function they are created in or else they are called global scoped.If we declare a var inside a function and try to call outside that fucntion, it wont work.

Eg: var a = 5;
var b;
console.log(a); //output 5
console.log(b); //output will be null

Let

It declares a variable in a block scope, that in is a set of opening and closing curly brackets. If we declare a let variable at the global scope, then redeclare it within a block, then the variable declared at the global scope will only be declared and not the one in the block scope or curly brackets.

Eg: let num=78;
console.log(num); //output 78
Let in a block scope:
{
let num=67;
console.log(num); //output 67
}

Const

Its value can only be assigned once and cannot be reassigned. It stays the same throught the program. Like the name suggests it is 'const(ant)'. If we try to reassign a const value it returns error message. It is also block scoped.

Eg: const this_one =10;
console.log(this_one); //output 10
