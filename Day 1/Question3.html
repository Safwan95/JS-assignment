Data Types In JS

Data types is what kind of data we store.Some of different kinds of data types are:
1.Number => It is usually to input number values like an integer, a floating point value, an exponential value.
2.String => It is to input group characters enclosed in single or double quotes like names of people, places etc.
3.Boolean => It is usually used for comparisons of logical conditions and returns value TRUE or FALSE.
4.Undefined => It means the data types is declared but doesnt contain any values. That is values are undefined.
5.NULL => The null value means no values.
6.Object => It contains key-value pairs in its address. Array and functions comes under this.
