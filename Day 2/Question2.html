Array Methods

1.copyWithin()=> Copies array elements within the array, to and from specified positions.
2.entries()=> Returns a key/value pair Array Iteration Object.
3.every()=> Checks if every element in an array pass a test.
4.fill()=> Fill the elements in an array with a static value.
5.filter()=> Creates a new array with every element in an array that pass a test.
6.find()=> Returns the value of the first element in an array that pass a test.
7.findIndex()=> Returns the index of the first element in an array that pass a test.
8.forEach()=> Calls a function for each array element.
9.from()=> Creates an array from an object.
10.includes()=> Check if an array contains the specified element.
11.indexOf()=> Search the array for an element and returns its position.
12.join()=> Joins all elements of an array into a string.
13.keys()=> Returns a Array Iteration Object, containing the keys of the original array.
14.map()=> Creates a new array with the result of calling a function for each array element.
15.pop()=> Removes the last element of an array, and returns that element.
16.reduce()=> Reduce the values of an array to a single value (going left-to-right).
17.reduceRight()=> Reduce the values of an array to a single value (going right-to-left).
18.some()=> Checks if any of the elements in an array pass a test .
19.sort()=> Sorts the elements of an array.
20.splice()=> Adds/Removes elements from an array.
21.valueOf()=> Returns the primitive value of an array.

String Methods

1.charCodeAt()=> Returns the Unicode of the character at the specified index.
2.endsWith()=> Checks whether a string ends with specified string/characters.
3.fromCharCode()=> Converts Unicode values to characters.
4.includes()=> Checks whether a string contains the specified string/characters.
5.localeCompare()=> Compares two strings in the current locale.
6.match()=> Searches a string for a match against a regular expression, and returns the matches.
7.repeat()=> Returns a new string with a specified number of copies of an existing string.
8.replace()=> Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.
9.search()=> Searches a string for a specified value, or regular expression, and returns the position of the match.
10.slice()=> Extracts a part of a string and returns a new string.
11.startsWith()=> Checks whether a string begins with specified characters.
12.substr()=> Extracts the characters from a string, beginning at a specified start position, and through the specified number of character.
13.substring()=> Extracts the characters from a string, between two specified indices.
14.toLocaleLowerCase()=> Converts a string to lowercase letters, according to the host's locale.
15.toLocaleUpperCase()=> Converts a string to uppercase letters, according to the host's locale.
16.trim()=> Removes whitespace from both ends of a string.
17.valueOf()=> Returns the primitive value of a String object.
